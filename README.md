# Kushagra Katiyar | Portfolio.v1

A personal portfolio website featuring an RX-7 with custom scroll interactions, dynamic tab switching, and responsive design. The background image is taken by me on Need for Speed 2015. There is also full mobile compatability. I took the picture on Need For Speed 2015.

Here it is: https://kushagrakatiyar06.github.io/Portfolio-v1/

# Tech Stack

Frontend: HTML5, CSS3 (Custom Properties, Flexbox, Grid). JavaScript

# Setup

Clone the repository.

Open index.html in a browser.

Ensure the assets/ directory is present to load icons and backgrounds.
